# Dockerised apt-cacher-ng

Apt-Cacher NG is a caching proxy for linux distribution packages, primarily Debian. https://www.unix-ag.uni-kl.de/~bloch/acng/

The container config adds support for alpine packages.

Build the image

`./make.sh build`

Run the image

`./make.sh run`

Rebuild and run 

`./make.sh rebuild`
